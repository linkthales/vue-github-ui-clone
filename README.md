## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/linkthales/vue-github-ui-clone.git
```

2. Install NPM packages

```sh
yarn
# or
npm install
```

3. Get the server up and running

```sh
yarn serve
# or
npm run serve
```
